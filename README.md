## 📋 About

A modern, fully internationalized (🇬🇧 / 🇫🇷) personal portfolio built with **React + Vite + TailwindCSS**. It showcases my expertise in full-stack development, UI/UX architecture, and web application engineering.

### Key Features

- 🌍 **i18n Ready** — English primary with French translation, manual toggle, and `localStorage` persistence
- 🐙 **Live GitHub Projects** — Dynamically fetches all public repositories from the GitHub API
- 📖 **README Viewer** — In-site viewer for project README files directly from GitHub
- 📥 **CV Download** — One-click PDF download of my resume
- 📊 **GitHub Activity Graph** — Visual contribution calendar showing my coding activity
- ✨ **Premium UI** — Dark theme, smooth animations, floating elements, fully responsive

## 🚀 Run Locally

```bash
# Install dependencies
npm install

# Create .env file with your GitHub token (optional)
echo "VITE_GITHUB_TOKEN=your_token_here" > .env

# Start dev server
npm run dev

# Build for production
npm run build
